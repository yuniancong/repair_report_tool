# 🚀 快速开始指南

## 10秒快速安装

```bash
# 1. 安装依赖（首次运行）
pip install customtkinter Pillow openpyxl reportlab

# 2. 启动程序
python launcher.py
```

## 📋 基本使用流程

### 1️⃣ 创建项目

1. 点击左侧边栏的 **"+ 添加"** 按钮
2. 在主区域输入项目描述（例如："电机维修"）

### 2️⃣ 添加图片

**方式一：点击按钮**
- 点击 **"📸 添加图片"** 选择图片文件

**方式二：批量添加**
- 点击 **"📁 批量添加"** 一次性选择多张图片

**方式三：拖放（如果支持）**
- 直接拖动图片文件到窗口中

### 3️⃣ 设置项目标题

- 在顶部输入框中输入项目标题
- 例如："2024年11月设备维修报告"

### 4️⃣ 导出文档

**导出 Excel:**
1. 点击右上角 **"📊 Excel"** 按钮
2. 选择保存位置
3. 等待生成完成

**导出 PDF:**
1. 点击右上角 **"📄 PDF"** 按钮
2. 选择保存位置
3. 等待生成完成

### 5️⃣ 保存项目

- 点击左下角 **"💾 保存"** 按钮
- 项目将保存为 JSON 文件
- 下次可以通过 **"📁 打开"** 加载

## 💡 使用技巧

### 项目管理

- **快速切换项目** - 直接点击左侧项目卡片
- **重命名项目** - 选中项目后，在主区域修改描述
- **删除项目** - 点击项目卡片上的 🗑 图标

### 图片管理

- **删除图片** - 点击图片卡片下方的"删除"按钮
- **查看原图** - 点击图片可查看完整尺寸（未来版本）
- **调整顺序** - 拖动图片重新排序（未来版本）

### 导出优化

- ✅ **设置标题** - 导出前务必设置项目标题
- ✅ **检查图片** - 确认所有图片都已正确加载
- ✅ **预览项目** - 使用预览功能检查内容
- ✅ **保存项目** - 导出前先保存项目，避免数据丢失

## ❓ 常见问题

### Q: 程序启动失败？

**A:** 确保已安装 CustomTkinter：

```bash
pip install customtkinter
```

### Q: 拖放功能不可用？

**A:** 需要安装 tkinterdnd2（可选）：

```bash
# macOS
brew install tkdnd
pip install tkinterdnd2

# Linux
sudo apt-get install python3-tk
pip install tkinterdnd2
```

不安装也可以正常使用，只是无法拖放图片。

### Q: 导出的 Excel/PDF 没有图片？

**A:** 检查：
1. 图片文件是否存在（未被移动或删除）
2. 图片路径是否正确
3. 是否有足够的磁盘空间

### Q: 中文显示为方块？

**A:** 确保系统已安装中文字体：

```bash
# Linux (Ubuntu/Debian)
sudo apt-get install fonts-wqy-microhei fonts-wqy-zenhei
```

Windows 和 macOS 默认支持中文。

### Q: 如何从旧版本迁移？

**A:** 新版本完全兼容旧版本的 JSON 项目文件：

1. 在旧版本中保存项目为 JSON
2. 在新版本中使用"打开"功能加载
3. 所有数据和图片路径都会保留
